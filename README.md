# Hospital Management System 🏥

Ye ek Python Flask project hai jo hospital ke patients aur appointments manage karne ke liye banaya gaya hai.

### ✨ Features:
* **Patient Registration:** Naye patients ko register karein.
* **Appointment Booking:** Doctor ke saath appointment set karein.
* **Login System:** Secure access ke liye.
* **Dashboard:** Sabhi records ko ek jagah dekhne ke liye.

### 🚀 Kaise chalayein?
1. Python install karein.
2. Terminal mein `pip install flask` likhein.
3. `python app.py` command se run karein.
